## Atelier Application IA

Application Next.js décrivant une feuille de route complète pour lancer un produit alimenté par l'intelligence artificielle. Le site propose :

- une vision en 3 phases (cadrage, prototype, industrialisation) ;
- une architecture de référence Next.js + orchestrateurs IA ;
- un planificateur interactif qui adapte les recommandations selon la maturité du projet ;
- une sélection de ressources et une FAQ pour passer à l'action.

## Démarrage local

```bash
npm install
npm run dev
```

Ouvrez ensuite [http://localhost:3000](http://localhost:3000) pour visualiser l'interface.

## Construction & tests

```bash
npm run lint
npm run build
```
